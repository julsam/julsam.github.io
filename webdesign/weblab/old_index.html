Les différentes étapes du Web Design :

Développement

Le développement de votre site peut se faire de plusieurs manières selon vos besoins.

Un site dit "statique" qui ne sera pas amené à évoluer régulièrement et sera donc parfait pour un site vitrine, de présentation.

Un site "dynamique" est un site dont le contenu est amené à évoluer régulièrement et qui, via une interface d'administration, vous pouvez permet par exemple :

  • d'ajouter, modifier ou supprimer du contenu
  • création de comptes pour vos utilisateurs
  • vente en ligne
  • blog
  • etc ...

Les possibilités sont infinis et les solutions pour répondre à vos besoins sont nombreuses.
Nous utilisons pour cela des CMS de type Joomla, Wordpress ou Dotclear.

Hébergement & Nom de domaine

Nous réservons le nom de domaine de votre site ( par exemple: www.votrecommerce.com ) que nous aurons auparavant définit avec vous.

Nous mettons ensuite en place son hébergement pour une durée déterminée avec vous.

Le site terminé est donc complètement fonctionnel et accessible par tout le monde depuis internet.

Référencement

Un site internet doit être facilement accessible depuis internet et de nos jours 90% des recherches se font via des moteurs de recherche (Google, Yahoo, Bing, etc...).
Apparaître en bonne position lors de ces recherches est donc un atout majeur pour la visibilité de votre site auprès du public.

La qualité du référencement de votre site dépends de critères auxquels nous accordons une grande importance : format des titres des pages, format des URL, code HTML valide et sémantique, soumission auprès des moteurs et annuaires de recherche.
